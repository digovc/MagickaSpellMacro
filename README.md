# Magicka Spell Macro

### Keys:

- To elements and shortcuts use the same key in keyboard: **Q, W, E, R, A, S, D, F, 1, 2, 3 and 4**;
- Mouse middle down: **MD**;
- Mouse middle up: **MU**;
- Mouse right down: **RD**;
- Mouse right up: **RU**;
- Shift down: **SD**;
- Shift up: **SU**;
- Time (1000 miliseconds): **TM**;
